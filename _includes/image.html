{% comment %} Example usage: {% include image.html file="someimage.png" url="http://verovio.humdrum.org" alt="VHV main page" caption="Text editor on the left and notation editor on the right" margin-bottom="-20px" %} Required parameters: file == URL of the image to display Optional parameters: url == URL to go to if image is clicked (otherwise, the image will be opened in a separate window/tab. alt == alternate text if image cannot be displayed. caption == text describing the figure, displayed below the image. Optional Parameters with default values: max-width == maximum width of the image (default is 75% of the text width) shadow="false" == do not draw a shadow under the image. By default shadows are displayed under images. margin-bottom == margin of the bottom of the image. Use negative values to tighten up the spacing between the image and the caption. By default the bottom margin is set to 0%. {% endcomment %}
